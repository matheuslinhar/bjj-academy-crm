# 🥋 BJJ Academy CRM

Sistema simples de CRM para academias de Jiu-Jitsu.

Este projeto foi criado para praticar desenvolvimento **Fullstack com Node.js**, criando uma aplicação para gerenciar alunos de uma academia.

---

## 🚀 Funcionalidades

- Cadastrar aluno
- Listar alunos
- Editar aluno
- Excluir aluno
- Buscar aluno
- Data de matrícula

---

## 🛠 Tecnologias utilizadas

- Node.js
- Express
- HTML
- CSS
- JavaScript
